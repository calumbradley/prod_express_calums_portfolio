# Express portfolios

This is a production ready build of my portfolio site ready for deployment with Node JS and express

### 1. Clone the project
```shell
git clone https://github.com/calumbradley/prod_calums_portfolio
```

### 2. Run the project
```shell
npm install
```

### 3. Install prod dependancies
```shell
npm start
```

### 3. Build the project
```shell
npm run build
```
